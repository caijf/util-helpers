formatDate(date, format) → {String}

Source:
  • formatDate.js, line 20
Since:
  • 1.1.0
格式化日期时间 y-年 M-月 d-日 h-小时 m-分钟 s-秒 q-季度 S-毫秒 该方法只做格式化，存在时区问题。如只传入年月日，但格式带有时间，会导致时区错误！如果需要更多时间操作，请使用 dayjs、moment等时间处理库
Example
const d1 = Date.now();
formatDate(d1); // => 2019-06-19 19:24:05
fotmatDate(d1, 'yyyy-MM-dd'); // => 2019-06-19
formatDate(d1, 'hh:mm'); // => 19:24
Parameters:
Name Type Default Description
date Date | String | Number 日期时间 默认 当前时间
format String yyyy-MM-dd hh:mm:ss 日期时间格式 默认格式 yyyy-MM-dd hh:mm:ss
Returns:
格式化的日期时间

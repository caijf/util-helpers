util-helpers

util-helpers 是一个基于业务场景的工具方法库，支持 umd （即 es6、amd、cmd 等都可使用）。如果你需要的是基础工具方法，请使用 Lodash、Moment、Dayjs

使用前请阅读文档，注意是否符合你的业务逻辑。

使用

安装依赖

npm install util-helpers --save

引入依赖

import { checkPhoneNumber } from 'util-helpers'

也可以按需引入

该方式只加载用到的方法

import checkPhoneNumber from 'util-helpers/lib/checkPhoneNumber'
